EXPO CLI

Установки 

expo cli 
npm install --global expo-cli

библиотека стилей css
yarn add styled-components 

axios делать запоосы на бэкенд
npm add axios
или
yarn add axios

//------------------------------------------------------------------------------

Запуск

Запуск Metro Bundle. Работа в web и на реальном устройстве через EXPO GO 
npm run start 
или 
yarn start

Остановка проекта
Cntrl+C

//------------------------------------------------------------------------------

Структура файлов 
1. package.json - Основные команды и библиотеки
2. babel.config.json - Конфигурация babel
3. app.json - Настройки приложения для expo (Есть React Native а поверх надстройка expo)
Для настройки expo, файл app.json 

4. App.json - Основной рабочий файл 
React Native есть запуск приложения происходит из App.js
